# Jim's Music Bot
A basic Discord Music bot written in Node.js.

# How To Use
Download the zipped file and in the settings.json input your own YouTube API key and own Discord Token.

## Packages Installed
0 - Discord.js   
1 - ffmpeg-binaries  
2 - get-youtube-id   
3 - iterate-js  
4 - upusscript  
5 - youtube-info  
6 - ytdl-core
